# 🏠 Jeedom Admin Toolkit

Une collection d'outils d'administration et de monitoring pour Jeedom.

## 📦 Contenu

| Outil | Description |
|---|---|
| [Monitor](./monitor/) | Surveillance des équipements — détecte les pannes, batteries faibles et pertes de contact |
| [Heartbeat](./heartbeat/) | Surveillance externe via Healthchecks.io — alerte si Jeedom tombe ou freeze |

## 🔧 Prérequis

- Jeedom 4.4+
- Plugin Telegram configuré avec une commande d'envoi
